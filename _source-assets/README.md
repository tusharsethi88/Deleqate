# _source-assets

Raw / source design material for Deleqate. **Nothing here is served by the app** —
it is the editable source kept for reference and future exports.

The app serves images from two code-wired locations (do **not** move these):

- `static/img/` — Django backend static assets
- `frontend/public/img/` — React/Vite public assets

## Contents

- `logo/` — brand logo source files (raw `.jpeg`/`.png`/`.mp4`, animations, screenshot).
- `site-assets/` — marketing/video moodboard material (characters, scenes, 16:9 samples).
- `web-ready-hero/` — exported hero image variants (1920 / 2752 px, before/after).

## History

Consolidated from previously loose folders that were scattered at the workspace root
(`LOGO`, `SKU Task Holders`) and inside the repo (`site assets`, `web-ready-hero`).
